# Vows and Views 🕊️
### An Interactive Digital Command Center for the Modern Couple

**Vows and Views** is a premium, single-file web application designed for couples who want to plan with intention. Moving away from the "toxic positivity" and clutter of traditional spreadsheets, this app focuses on a grounded experience that bridges the physical wedding with a mindful digital presence.

## 🌿 Key Features
- **Social Media Hub:** Curate your wedding "Grid," track hashtags, and store vendor handles for seamless day-of tagging.
- **Smart Budgeting:** A grounded view of finances including service fees and tips.
- **Ivory, Sage, & Gold Aesthetic:** Designed to match the **Simplify to Glorify** brand identity.
- **Privacy First:** All data is saved locally on the user's browser—no accounts or data collection required.

## 🛠️ Tech Stack
- **HTML5/CSS3** (Tailwind CSS)
- **Vanilla JavaScript** (LocalStorage for data persistence)
- **Typography:** Playfair Display & Inter

## 🚀 How to View
Simply open `index.html` in any modern web browser.
