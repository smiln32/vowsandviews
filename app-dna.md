# THE DIGITAL AISLE | App DNA & Project Logic

## 1. BRAND CORE (Simplify to Glorify)
- **Voice:** Grounded, supportive, and non-judgmental. 
- **Style:** Artisanal minimalism. 
- **Mission:** Replacing stressful spreadsheets with an intentional "Digital Aisle" experience.
- **Anti-Pattern:** No toxic positivity. Keep micro-copy realistic and calm.

## 2. DESIGN PALETTE (STRICT)
- **Base:** Ivory (#F9F7F2) - Soft and high-end.
- **Primary:** Sage (#7A9482) - For action buttons and navigation.
- **Accent:** Gold (#C5A059) - For progress bars and premium success highlights.
- **Fonts:** 'Playfair Display' (Serif) for elegance; 'Inter' (Sans) for clean data.

## 3. TECHNICAL DNA
- **Architecture:** Single-file `index.html`.
- **Frameworks:** Tailwind CSS (Style) + Alpine.js (Smart Logic).
- **Privacy:** Data persists via `localStorage` ONLY. No cloud, no logins.
- **Goal:** A lightweight, "instant-open" dashboard for Etsy customers.

## 4. COMPONENT REGISTRY
- [x] Dashboard: Ivory card layout with Sage borders.
- [x] Budget: Gold-shimmer progress bar.
- [x] Timeline: Smart-calculated deadlines based on wedding date.
- [x] Social Hub: The "Grid" planner and Vendor tagging directory.
